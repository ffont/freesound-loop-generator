The Freesound Loop Generator is a newer version (2020 update) of the old FreeMaschine! hack developed by Javi Agenjo, Bram de Jong and Frederic Font at Barcelona Music Hack Day 2014 (see video here). It includes a bit (not too much) of code refactoring and adds export functionality so you can download the loops and sequences you create. More info (including source code) is available here.
